# Data Privacy Policy

## Data Collection
- User registration information
- Report submissions and images
- Location data (with consent)
- Usage analytics

## Data Processing
- Automatic face blurring in uploaded images
- AI analysis for content classification
- Anonymization of sensitive data

## Data Retention
- User data: Retained until account deletion
- Reports: Retained for 5 years
- Images: Face-blurred versions stored

## User Rights
- Access to personal data
- Data correction and deletion
- Opt-out of analytics

## Security Measures
- Encryption in transit and at rest
- Regular security audits
- Access controls and monitoring
